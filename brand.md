# Brand — Trade Arena

_Status: active_

Trade Arena uses a playful minimal system with darker pastel depth, sharp product edges, and bright gamified accents. The chart stays dominant while the surrounding UI reads as light, fast, and consumer-friendly instead of dense or institutional.

## Direction

- Brand name: `Bubblegum Signal`
- Tone: `playful`, `minimal`, `consumer-tech`, `lightly gamified`
- Visual rule: strong contrast where actions matter, quiet contrast everywhere else

## Typography

- Sans: `Geist`
- Serif fallback role: `Geist`
- Mono: `Geist Mono`

## Surface Language

- Tight radius system with `0.625rem` base corners
- Soft pastel fields with crisp interaction contrast
- Minimal shadow use; separation comes from tone shifts and borders first
- Buttons and interactive labels should always prefer readable foreground contrast over tonal subtlety

## Usage Notes

- Use semantic tokens only: `bg-card`, `text-foreground`, `text-muted-foreground`, `border-border`, `bg-primary`, `text-primary-foreground`
- Keep decorative gradients restrained; the product should feel clean before it feels expressive
- Reserve the hottest pink primary moments for actions, active states, and competitive signals
- Charts, execution states, and agent outcomes should remain the visual focus rather than surrounding chrome
